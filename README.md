# Kite Drive - 自动驾驶车位预约与支付系统

> 🏆 **黑客松荣誉**：Casual Hackathon "SPARK AI" / Kite AI 支付赛道 / **季军（Third Prize）**

> 基于 AI 与智能合约的自动驾驶车辆自动车位预约与支付解决方案

## 🎯 项目愿景

在自动驾驶普及的未来，车辆寻找车位将面临新的挑战：人类驾驶员可以"将就"停车，但自动驾驶车辆需要精确的预定车位。Kite Drive 为自动驾驶车辆提供端到端的自动车位预约与支付服务，消除停车等待时间，优化城市停车资源。

## 🚀 核心功能

### 🅿️ 智能车位匹配
- **实时车位发现**：对接城市停车数据库，实时获取可用车位信息
- **AI 决策引擎**：基于车辆尺寸、停车时长、费用优化等多维度智能匹配
- **LLM 解释**：通过大语言模型生成可理解的停车决策解释

### 💳 自动化支付
- **AA 钱包集成**：使用 Kite-AI 的 AA（账户抽象）钱包实现无缝支付
- **智能合约执行**：通过智能合约自动执行停车费支付
- **支付凭证生成**：实时生成链上交易凭证与验证

### 🚗 端到端体验
- **一键预约**：车辆或用户可一键发起停车预约
- **状态追踪**：实时显示车辆前往、停车、离开全流程
- **历史记录**：完整的停车历史与支付记录

## 📱 使用演示

### 演示流程
1. **访问应用**：打开部署的网站
2. **查看车位**：系统显示可用车位地图
3. **发起预约**：点击"预约车位"按钮
4. **AI决策**：系统智能选择最优车位
5. **支付确认**：自动完成链上支付
6. **查看凭证**：显示交易哈希和停车凭证

### 演示视频
项目包含完整演示视频，展示自动驾驶车辆从寻找车位到完成停车的全流程。

## 🔧 技术栈

### 前端
- **HTML/CSS/JavaScript** - 基础前端技术
- **响应式设计** - 适配多端显示

### 后端
- **Node.js** - 服务器运行环境
- **Vercel Functions** - 无服务器函数
- **Express** - Web框架（本地开发）

### AI与区块链
- **DashScope API** - 阿里云大语言模型服务
- **Kite-AI SDK** - AA钱包与支付集成
- **智能合约** - 自动执行支付逻辑
